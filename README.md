# 🛒 AI Smart Supermarket Management System

> An IoT-enabled Smart Supermarket platform that combines **React**, **Spring Boot**, **PostgreSQL**, **ESP32**, and **RFID technology** to provide a seamless cashier-less shopping experience with real-time inventory management and analytics.

# 📖 Overview

The AI Smart Supermarket Management System is a full-stack web application integrated with IoT hardware that enables customers to shop using smart trolleys while allowing administrators to monitor inventory, sales, and trolley activity in real time.

The system eliminates manual billing by automatically identifying products through RFID technology and generating bills digitally.
